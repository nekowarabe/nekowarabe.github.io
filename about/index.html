多句話簡介

於此 Blog 自稱為 Nekowarabe ，現居台灣。

平時喜歡隨便亂逛網站，碰到感興趣的東西就廢寢忘食的研究，通常感興趣的都是程式相關的技術，因為我不喜歡只知道如何使用，而傾向要透徹了解其原理或概念。

如前述故學習速度很慢，常常看到原理 A 使用到概念 B 與技術 C ，得額外花時間去搞懂，而它們可能又各自有使用到其它⋯⋯ Loop 。

現在主要技能都點在後端，比較熟悉的語言為：

  • Go
  • Java
  • Dart

但我實際用在工作上的只有 Go ，其它兩個除了 Dart 用於 Flutter ； Java 只剩無聊時會拿出來寫寫，連 Spring 都不會。
